# AI 營運顧問系統 — 引導流程與遊戲化設計

**建立日期**：2025-01-27  
**版本**：v1.0

---

## 📋 設計目標

### 核心問題
1. **頁面太長**：Dashboard 一次顯示所有模組，資訊過載
2. **缺乏引導**：使用者不知道從哪裡開始、如何操作
3. **缺乏動機**：使用者可能不清楚系統價值，缺乏持續使用的動力

### 解決方案
1. **Step-by-step 引導**：將長頁面拆解成步驟，逐步引導
2. **遊戲化機制**：加入積分、成就、等級、進度條
3. **互動式教學**：遮罩層 + 工具提示，高亮重點區域
4. **漸進式揭露**：首次使用時隱藏部分功能，完成引導後逐步開放

## 🎯 引導流程設計（Onboarding Flow）

### 階段 1：歡迎與目標設定（首次登入）
```
步驟 1：歡迎頁面
- 展示產品價值：「讓數據變成行動」
- 快速介紹：3 個核心功能（Dashboard、Guideline、任務）

步驟 2：目標設定
- 詢問使用者角色（行銷人員 / 營運經理 / 品牌老闆）
- 設定主要關注指標（營收 / 流量 / 轉換率）
- 預期達成的目標（提升 X%）

步驟 3：完成引導承諾
- 「完成引導即可獲得 100 積分 + 新手成就徽章」
```

### 階段 2：核心功能教學（Step-by-step）

#### Step 1：認識 Dashboard（2-3 分鐘）
**目標**：讓使用者理解如何快速掌握營運狀況

```
1. 高亮「營運摘要區」
   - 工具提示：「這裡可以看到本週總營收、流量、轉換率」
   - 互動：點擊任一卡片查看詳細說明

2. 高亮「KPI 金字塔」
   - 工具提示：「營收 = 流量 × 轉換率 × 客單價，點擊任一指標可展開」
   - 互動：點擊「流量」指標，展開詳細說明

3. 完成任務：點擊任一 KPI 指標
   - 獎勵：+10 積分
```

#### Step 2：理解 Guideline 建議（3-4 分鐘）
**目標**：讓使用者學會如何從 AI 建議獲得行動方向

```
1. 高亮第一個 Guideline 卡片
   - 工具提示：「這是 AI 根據數據異常自動生成的建議」
   - 展開說明：展示「觀察結果」和「建議行動」

2. 互動：閱讀第一個 Guideline 的完整內容

3. 引導操作：點擊「＋ 加入任務清單」
   - 工具提示：「將建議轉為可追蹤的任務」
   - 示範：自動打開任務建立 Modal（預填資料）

4. 完成任務：成功建立第一個任務
   - 獎勵：+20 積分 + 「行動實踐者」成就
```

#### Step 3：管理任務清單（2-3 分鐘）
**目標**：讓使用者學會追蹤與管理任務

```
1. 高亮「行動任務清單」區塊
   - 工具提示：「這裡可以看到所有待執行任務」

2. 互動：點擊剛建立的任務

3. 引導操作：切換任務狀態（未開始 → 進行中）
   - 工具提示：「更新任務狀態，系統會追蹤執行進度」
   - 示範：狀態切換動畫

4. 完成任務：成功更新任務狀態
   - 獎勵：+15 積分
```

#### Step 4：探索進階功能（選修）
**目標**：引導使用者探索其他重要功能

```
1. 高亮「七大流量策略表」
   - 工具提示：「點擊任一流量來源查看詳細分析」
   - 互動：點擊「搜尋」展開詳細資訊

2. 高亮「轉換漏斗」
   - 工具提示：「追蹤使用者從瀏覽到購買的轉換流程」

3. 完成任務：點擊查看流量來源詳情
   - 獎勵：+10 積分
```

### 階段 3：完成引導慶祝
```
1. 顯示完成引導動畫
   - 彩帶動畫
   - 顯示獲得的積分與成就

2. 解鎖所有功能
   - 「現在你可以完整使用系統了！」

3. 引導下一步
   - 「每週一查看 Dashboard，追蹤任務進度」
   - 「完成任務可獲得積分與成就」
```

## 🎮 遊戲化機制設計（Gamification）

### 1. 積分系統（Points System）

#### 積分獲得方式
| 動作 | 積分 | 說明 |
|------|------|------|
| 完成引導 | +100 | 首次完成完整引導 |
| 建立任務 | +20 | 從 Guideline 建立任務 |
| 完成任務 | +50 | 標記任務為「已完成」 |
| 任務有成效 | +30 | 任務完成後 KPI 改善 >5% |
| 每日登入 | +5 | 連續登入有額外加成 |
| 查看 Guideline | +2 | 展開查看 Guideline 詳細內容 |
| 更新任務狀態 | +10 | 更新任務狀態（進行中等） |
| 匯出報告 | +15 | 匯出報告或傳送至 LINE |

#### 積分用途
- 顯示在個人檔案
- 解鎖新功能（未來擴展）
- 排行榜（可選，團隊模式）

### 2. 成就系統（Achievement System）

#### 成就列表

**新手成就**
- 🎓 **初學者**：完成引導流程
- 📊 **數據觀察家**：查看 10 個 Guideline 建議
- ✅ **行動實踐者**：建立第一個任務
- 🏆 **任務達人**：完成 5 個任務

**進階成就**
- 💎 **效率大師**：一週內完成 10 個任務
- 📈 **營運專家**：任務平均成效提升 >10%
- 🔥 **持續精進**：連續 4 週每週登入
- ⭐ **完美執行**：任務完成率 100%（10 個以上）

**特殊成就**
- 🎯 **精準行動**：完成任務後成效提升 >20%
- 🌟 **團隊協作**：與顧問互動 5 次以上
- 🚀 **快速上手**：首次登入後 24 小時內完成 3 個任務

#### 成就展示
- 在 Dashboard Header 顯示當前成就
- 點擊可查看完整成就列表
- 獲得成就時顯示慶祝動畫

### 3. 等級系統（Level System）

#### 等級設計
```
等級 1：新手營運者（0-100 積分）
- 解鎖：基本 Dashboard、Guideline、任務功能

等級 2：數據觀察者（101-300 積分）
- 解鎖：進階圖表、流量來源詳情

等級 3：行動實踐者（301-600 積分）
- 解鎖：任務成效分析、個人報告

等級 4：營運專家（601-1000 積分）
- 解鎖：Benchmark 比較、自訂儀表板

等級 5：營運大師（1001+ 積分）
- 解鎖：所有功能、優先客服支援
```

#### 等級視覺化
- 進度條顯示：當前積分 / 下一等級所需積分
- 等級徽章：顯示在個人檔案
- 升級動畫：達到新等級時慶祝

### 4. 進度追蹤（Progress Tracking）

#### 週期性目標
- **每週目標**：完成 3 個任務、查看 5 個 Guideline
- **每月目標**：完成 15 個任務、提升營收 X%
- **季度目標**：維持 80% 任務完成率

#### 進度顯示
- Dashboard 右上角顯示「本週進度」
- 進度條 + 百分比
- 完成目標時顯示通知

## 🎨 UI/UX 設計細節

### 引導遮罩層（Overlay）
```
- 半透明黑色背景（rgba(0,0,0,0.7)）

- 工具提示：氣球樣式，指向高亮區域

```
[●] [○] [○] [○] 步驟 1 / 4

## 📱 互動流程範例

### 首次登入流程

```
1. 使用者登入
   ↓
2. 檢測到首次登入 → 顯示歡迎 Modal
   ↓
3. 使用者點擊「開始引導」
   ↓
4. 進入 Step 1：認識 Dashboard
   - 遮罩層出現
   - 高亮「營運摘要區」
   - 顯示工具提示
   ↓
5. 使用者點擊「下一步」
   ↓
6. 高亮「KPI 金字塔」，引導互動
   ↓
7. 使用者完成互動 → 獲得 +10 積分通知
   ↓
8. 自動進入 Step 2：理解 Guideline
   ...
   ↓
9. 完成所有步驟 → 顯示完成慶祝
   - 總積分：+100
   - 獲得成就：「初學者」
   ↓
10. 解鎖完整功能，正常使用
```

- 週期性目標激勵使用者每週完成任務
- 成就系統讓使用者有收集動機
- 等級系統提供長期成長目標
