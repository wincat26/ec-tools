# 電商營運輔助工具 - Sitemap 規劃

## 🏠 主要頁面架構

### 1. Portal 頁面（首頁）
**路徑：** `/`  
**功能：** 電商登頂儀表板  
**內容：**
- Hero Section（7級等級系統展示）
  - 當前等級：新手村民 → 電商之神
  - 經驗值進度條
  - 升級所需條件
- 營收目標進度
- 缺口分析概覽
- 今日任務（依等級解鎖）
- 快速行動按鈕

---

### 2. 目標設定頁面
**路徑：** `/goal-setting`  
**功能：** 營收目標設定與拆解  
**內容：**
- 目標輸入表單
- 歷史數據上傳
- 自動拆解預覽
- 目標確認

---

### 3. 缺口分析頁面
**路徑：** `/gap-analysis`  
**功能：** 詳細缺口分析與診斷  
**內容：**
- 三因子詳細分析（T×C×A）
- 歷史趨勢圖表
- 競品對比
- 改善建議

---

### 4. 活動中心
**路徑：** `/campaigns`  
**功能：** 活動方案推薦與管理  
**內容：**
- 推薦活動卡片
- 活動篩選器（流量型/轉換型/客單價型）
- 活動效果預測
- 執行中活動狀態

---

### 5. 任務中心
**路徑：** `/tasks`  
**功能：** 任務管理與追蹤  
**內容：**
- 今日任務列表
- 本週任務規劃
- 任務歷史記錄
- 完成度統計

---

### 6. 成就系統
**路徑：** `/achievements`  
**功能：** 小老闆之路等級與成就管理  
**內容：**
- 7級等級路線圖（新手村民 → 電商之神）
- 當前等級詳細資訊
- 升級條件與進度
- 徽章收藏庫（營收/技能/社群成就）
- 等級特權展示
- 升級慶祝動畫

---

### 7. 數據中心
**路徑：** `/data`  
**功能：** 數據管理與同步  
**內容：**
- 數據上傳
- API 連接設定
- 數據驗證
- 同步狀態

---

### 8. 設定頁面
**路徑：** `/settings`  
**功能：** 個人與系統設定  
**內容：**
- 品牌資訊
- 通知設定
- API 金鑰管理
- 帳戶設定

---

## 🔄 頁面流程設計

### 新用戶流程
```
Landing Page → 目標設定 → 數據上傳 → 缺口分析 → 活動選擇 → Portal 儀表板
```

### 日常使用流程
```
Portal 儀表板 → 任務中心 → 活動執行 → 數據檢視 → Portal 儀表板
```

### 深度分析流程
```
Portal 儀表板 → 缺口分析 → 數據中心 → 活動中心 → 任務中心
```
