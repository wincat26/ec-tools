## 📊 資料匯入選項設計

### 選項 1: CSV 檔案上傳（最簡單）
```javascript
const csvTemplate = {
  columns: ['日期', '營收', '流量', '轉換率', '客單價'],
  example: [
    '2024-01-01,450000,12500,2.3,1580',
```

**優勢：**
- 立即可用，無需API設定
- 適合所有電商平台
- 用戶完全控制數據

### 空狀態設計原則
- **明確指引**：清楚告訴用戶下一步要做什麼
- **價值預覽**：展示匯入數據後能獲得什麼功能
- **降低門檻**：提供最簡單的開始方式
- **即時回饋**：匯入過程中顯示進度

## 🔄 資料同步策略

### 初次匯入
1. **手動上傳** CSV → 立即分析
2. **API 連接** → 授權 → 首次同步
3. **數據驗證** → 格式檢查 → 錯誤提示

### 持續同步
- **每日自動同步**：API 連接的平台
- **手動更新**：CSV 上傳用戶
- **增量更新**：只同步新數據

### 數據品質控制
```javascript
const dataValidation = {
  required: ['date', 'revenue'],
  optional: ['traffic', 'cvr', 'aov'],
  validation: {
    revenue: 'number > 0',
    traffic: 'number >= 0', 
    cvr: 'percentage 0-100',
    aov: 'number > 0'
  }
}
```
