# AI 營運顧問系統 - Prototype Sitemap 設計

## 整體架構總覽

```
AI 營運顧問系統
├── 1. Dashboard (總覽)
├── 2. 數據分析 (Data)
├── 3. 洞察中心 (Insights)
├── 4. 行動方案 (Actions)
├── 5. 角色/顧問池 (Consultants)
```

## 詳細分區規劃

### 1. Dashboard (總覽) - `/dashboard.html`

**核心定位**：一頁式最重要聚焦，設入口

**主要內容**：
- **總營收 KPI 卡片**
  - 總營收（點擊可展開分解）
  - 流量（點擊可展開七大流量策略表）
  - 轉換率（點擊可展開轉換漏斗）
  - 平均訂單金額（點擊可展開拆解）
  
- **本週重點異常警示**
  - 異常 KPI 紅色警示
  - 點擊異常指標 → 自動跳轉洞察中心
  
- **核心指標趨勢**
  - 簡化版趨勢圖表（7天/30天切換）
  
- **快速導入 3 大單元**
  - 快速入口按鈕：
    - 📊 查看數據分析
    - 💡 查看洞察建議
    - ✅ 管理行動方案

**互動流程**：
- 點擊異常指標 → 跳轉洞察中心（自動帶入異常項目）
- 點擊 KPI 卡片 → 展開詳細分解（內聯展開，不跳頁）
- 快速入口按鈕 → 跳轉對應分區

---

### 2. 數據分析 (Data) - `/data.html`

**核心定位**：可 drill-down 的主題式數據分析

**主要內容**：
- **詳細 GMV 拆解**
  - 總營收公式：全營收 - 取消退貨 = 總營收
  - 取消率計算與趨勢
  - 訂單筆數分析
  
- **指標對比**
  - 上週對比（WoW）
  - 去年同期對比（YoY）
  - 趨勢圖表
  
- **多維度分析分頁（Tab）**
  - 📊 流量分析
    - 七大流量策略表
    - 各流量來源會員結構
    - 流量來源轉換率觀察
    
  - 📦 商品分析
    - 熱銷商品 Top 10
    - 商品分類表現
    - 單價分佈
    
  - 📢 廣告分析
    - 廣告成效總覽
    - 各平台表現對比
    - 廣告 ROI 分析
    
  - 👥 會員分析
    - 新舊會員占比
    - 會員生命週期價值（LTV）
    - 會員轉換漏斗

**互動流程**：
- 發現問題 → 點擊「查看洞察」按鈕 → 跳轉洞察中心（帶入相關數據）
- Drill-down：點擊各項數據可深入查看更多細節

---

### 3. 洞察中心 (Insights) - `/insights.html`

**核心定位**：動靜態洞察區，聚合 AI 與專家

**主要內容**：
- **AI 觀察摘要**
  - 本週 AI 重點觀察（卡片式）
  - 異常診斷結果
  - 趨勢預測
  
- **智能異常診斷**
  - 自動偵測異常指標
  - 異常原因分析
  - 影響範圍評估
  
- **專家/角色洞察**
  - 各流量來源專家建議（需訂閱）
  - 角色解讀（選擇特定角色解讀數據）
  - 外部顧問建議
  
- **高階趨勢解讀**
  - 長期趨勢分析
  - 季節性模式識別
  - 競爭對比洞察

**互動流程**：
- 數據分析發現問題 → 跳轉此頁（自動帶入問題）
- 選擇「由哪個角色解讀」→ 角色對話視窗
- 點擊「加入任務」→ 自動轉入行動方案
- 點擊「採納建議」→ 標記為已處理

**特殊功能**：
- 角色切換按鈕（在洞察卡片下方）
- 「跟該領域專家對話」按鈕
- 一鍵加入任務清單

---

### 4. 行動方案 (Actions) - `/actions.html`

**核心定位**：與 task 清單連動的執行管理

**主要內容**：
- **行動清單管理**
  - 任務列表（未開始/進行中/已完成）
  - 任務優先級標記
  - 任務指派與時間線
  
- **建議優先級**
  - AI 建議的優先級排序
  - 建議來源標記（AI/專家/角色）
  
- **任務模板/推薦**
  - 常見任務模板庫
  - AI 推薦的任務模板
  
- **進度追蹤**
  - 任務完成度圖表
  - 里程碑追蹤
  - 預期 vs 實際進度

**互動流程**：
- 從洞察中心加入任務 → 自動出現在此頁
- 點擊任務 → 展開詳細資訊與執行步驟
- 標記完成 → 更新進度與 Dashboard 指標
- 從模板建立任務 → 快速建立標準化任務

---

### 5. 角色/顧問池 (Consultants) - `/consultants.html`

**核心定位**：支援外部諮詢及人格化技能

**主要內容**：
- **角色卡選擇**
  - Rosebud（資料分析師）
  - 挑戰者（質疑策略的角色）
  - 智者（提供深度見解）
  - 其他角色...
  
- **不同角色發問/切換**
  - 角色對話介面
  - 角色專屬技能包展示
  - 切換角色按鈕
  
- **顧問個人化分析**
  - 各角色的分析視角
  - 角色推薦的見解
  - 外部專家顧問（訂閱制）

**互動流程**：
- 選擇角色 → 進入對話模式 → 提問獲得回答
- 在洞察中心選擇「由 X 角色解讀」→ 顯示角色視角的分析
- 訂閱外部專家 → 獲得專家專屬建議

---

### 6. 報告設定&歷史 (Reports) - `/reports.html`

**核心定位**：行政+歷史性功能

**主要內容**：
- **自動推送設定**
  - 週報/月報自動發送
  - LINE Notify 設定
  - Email 報告設定
  
- **週報/月報下載**
  - 報告歷史列表
  - PDF 下載功能
  - 報告預覽
  
- **訂閱管理**
  - 專家訂閱狀態
  - 報告訂閱設定

**互動流程**：
- 設定自動推送 → 選擇頻率與接收方式
- 查看歷史報告 → 下載或預覽
- 管理專家訂閱 → 跳轉專家訂閱頁

**核心定位**：偏後台行為

**主要內容**：

  - API 金鑰管理

## 使用者動線典型路徑

### 日常追蹤（高頻簡潔型）

```
登入 → Dashboard 
  → 看到異常指標（紅色警示）
  → 點擊異常指標 
  → 跳轉洞察中心（自動帶入異常項目）
  → 查看 AI 診斷與專家建議
  → 點擊「加入任務」
  → 自動轉入行動方案
  → 在行動方案追蹤執行進度
```

### 專題診斷（深度探索型）

```
特定需求 → 數據分析區
  → 選擇分頁（流量/商品/廣告/會員）
  → 深入分析發現問題
  → 點擊「查看洞察」
  → 跳轉洞察中心
  → 選擇角色解讀或召喚外部顧問
  → 取得建議與行動方案
  → 建立/調整行動方案
  → 行動區跟蹤執行
```

### 策略規劃/目標管理

```
Dashboard 目標進度條/提醒
  → 點擊調整目標
  → 行動方案區
  → 拆解為多個子目標
  → 分派成員與時間線
  → 定期追蹤
  → 自動報告推送
```

### AI/專家對話學習

```
角色/顧問池
  → 切換角色
  → 在洞察中心提問、互動
  → 拓展專業技能包
  → 取得模版建議
```

## 頁面間跳轉設計原則

1. **上下文保持**：從某頁跳轉到另一頁時，帶入相關上下文（如異常項目、數據篩選條件）
2. **一鍵操作**：關鍵操作提供快速入口（如「加入任務」）
3. **返回路徑**：每個頁面提供清晰的返回路徑
4. **麵包屑導航**：深度頁面提供麵包屑，清楚顯示當前位置
