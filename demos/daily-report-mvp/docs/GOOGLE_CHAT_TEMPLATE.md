## 📋 模板結構總覽

### 核心設計理念

每日數據彙整日報的 Google Chat 卡片應包含以下核心概念：

1. **營收公式拆解**：流量 × 轉換率 × 客單價
2. **趨勢分析**：週變化、目標達成進度

## 🎨 卡片結構設計

### 卡片標題區（Header）
```
📊 每日數據彙整日報
日期：YYYY-MM-DD
```

### 區塊 1：當日關鍵指標（核心營收公式）

**標題**：📈 當日關鍵指標

**內容**：
1. **總營收**（主指標）
   - 金額顯示
   - vs. 上週同期變化（% + 圖示）
   - 顏色：上升綠色 / 下降紅色

2. **營收公式拆解**（三個組成要素）
   - **流量（Sessions）**
     - 當日 Sessions
     - vs. 上週同期變化
   - **轉換率（CVR）**
     - 當日 CVR
     - vs. 上週同期變化
   - **客單價（AOV）**
     - 當日 AOV
     - vs. 上週同期變化

3. **訂單數**
   - 當日訂單數
   - 平均客單價

---

### 區塊 2：廣告表現

**標題**：💰 廣告表現

**內容**：
1. **廣告花費**
   - 當日廣告花費

2. **ROAS**
   - 當日 ROAS
   - 顏色指標（≥3.0 綠色 / ≥2.0 黃色 / <2.0 紅色）

3. **廣告效率分析**（可選）
   - 廣告花費 vs. 營收占比
   - ROAS 趨勢

---

### 區塊 3：當月目標達成

**標題**：🎯 當月目標達成

**內容**：
1. **目標達成率**（主指標）
   - 百分比顯示
   - 顏色：≥80% 綠色 / ≥50% 黃色 / <50% 紅色
   - 進度條視覺（如果 Google Chat 支援）

2. **月迄今營收**
   - MTD 營收
   - 預估當月營收（根據當前進度推算）

3. **當月目標**
   - 目標金額
   - 剩餘差距（目標 - MTD 營收）

4. **達成進度分析**
   - 已過天數 / 當月總天數
   - 每日平均需達成金額（以達成目標）

## 📊 數據顯示規範

### 數字格式化規則

1. **貨幣格式**
   - ≥10,000：顯示為「X.X 萬」（例如：$2.5 萬）
   - <10,000：顯示為「$X,XXX」（例如：$8,500）

2. **百分比格式**
   - 保留 2 位小數（例如：1.50%）
   - 正數顯示「+」號（例如：+15.2%）
   - 負數顯示「-」號（例如：-10.5%）

3. **整數格式**
   - 使用千分位符號（例如：1,234）

### 顏色規範

| 狀態 | 顏色代碼 | 使用場景 |
|------|---------|---------|
| 優秀/上升 | `#51CF66` (綠色) | 達成率 ≥80%、ROAS ≥3.0、上升趨勢 |
| 良好/持平 | `#FFD93D` (黃色) | 達成率 50-80%、ROAS 2.0-3.0 |
| 需注意/下降 | `#FF6B6B` (紅色) | 達成率 <50%、ROAS <2.0、下降趨勢 |

### 圖示規範

| 趨勢 | 圖示 | 使用場景 |
|------|------|---------|
| 上升 | 📈 | 營收/CVR/Sessions 上升 |
| 下降 | 📉 | 營收/CVR/Sessions 下降 |
| 持平 | ➡️ | 變化 <5% |

## 🔄 營收公式拆解邏輯

### 公式
```
總營收 = Sessions × CVR × AOV
```

### 分析方式

1. **流量（Sessions）**
   - 當日 Sessions
   - vs. 上週同期變化
   - 影響：流量增加 → 營收增加（假設 CVR、AOV 不變）

2. **轉換率（CVR）**
   - 當日 CVR
   - vs. 上週同期變化
   - 影響：CVR 提升 → 營收增加（假設 Sessions、AOV 不變）

3. **客單價（AOV）**
   - 當日 AOV
   - vs. 上週同期變化
   - 影響：AOV 提升 → 營收增加（假設 Sessions、CVR 不變）

### 顯示方式

在 Google Chat 卡片中，營收公式的三個要素應並列顯示，讓使用者一目了然：
- 哪個要素提升了？
- 哪個要素下降了？
- 整體營收的變化是如何形成的？

---

## 📈 趨勢分析邏輯

### 週變化（Week-over-Week, WoW）

**計算方式**：
```
變化百分比 = (當日值 - 上週同期值) / 上週同期值 × 100%
```

**判斷標準**：
- 變化 ≥ 10%：顯著上升/下降
- 變化 5-10%：小幅上升/下降
- 變化 < 5%：持平

### 目標達成分析

**達成率**：
```
達成率 = MTD 營收 / 當月目標 × 100%
```

**預估當月營收**：
```
預估營收 = MTD 營收 / 已過天數 × 當月總天數
```

**每日平均需達成**：
```
每日需達成 = (當月目標 - MTD 營收) / 剩餘天數
```
